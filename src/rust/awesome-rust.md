# Awesome-Rust

## Library

- **tokio**

> 使用Rust编写可靠的异步应用程序的运行时

- **rayon**

> Rust的数据并行库

- **reqwest**

> 简单而强大的Rust HTTP客户端

- **clap**

> 一个功能齐全、快速的Rust命令行参数解析器

- **rusqlite**

> 符合人体工程学的SQLite for Rust绑定

- **hyper**

> Rust的HTTP库


## Tool

- **mdBook**

> 从markdown文件创建图书。类似于Gitbook，但在Rust中实现

- **hurl**

> 使用纯文本投掷、运行和测试HTTP请求

- **sshx**

> 通过Web快速、协作的实时终端共享

- **simple-http-server**

> Rust中的简单http服务器(Windows/Mac/Linux)
